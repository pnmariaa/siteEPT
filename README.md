# 🎓 Currículos 3º CPM

Plataforma desenvolvida para auxiliar os alunos do **3º Colégio da Polícia Militar do Paraná (3º CPM)** na criação de currículos, busca por oportunidades de estágio e emprego, além do acesso a cursos gratuitos para desenvolvimento profissional.

## 📌 Funcionalidades

- 🏠 Página inicial com apresentação do projeto e novidades.
- 📄 Formulário para criação de currículo.
- 💼 Página com vagas de estágio e emprego.
- 📚 Área com indicação de cursos e certificados gratuitos.
- 🤖 Chatbot "Nina" para atendimento e suporte aos usuários.
- 🌙 Modo claro/escuro.
- 📱 Interface responsiva e intuitiva.

## 🤖 Chatbot

A assistente virtual **Nina** foi desenvolvida para responder às principais dúvidas dos usuários, oferecendo informações como:

- Como utilizar o site;
- Como contratar um aluno;
- Como enviar um currículo;
- Informações sobre o 3º CPM;
- Informações sobre os turnos da escola.

Além disso, o chatbot possui:

- respostas automáticas;
- indicador de digitação;
- menu interativo;
- opção para finalizar a conversa;
- navegação por palavras-chave.

---

## 🌙 Modo Claro/Escuro

O projeto possui suporte à alternância entre tema claro e escuro, alterando automaticamente as cores da interface para melhorar a experiência do usuário.

---

## 📋 Páginas

### Página Inicial
Apresenta o projeto e exibe notícias e novidades.

### Criar Currículo
Permite que o aluno preencha seus dados pessoais, habilidades, experiências e objetivo profissional.

### Vagas
Lista oportunidades de estágio e emprego para os alunos.

### Certificados
Disponibiliza links para plataformas de cursos gratuitos que ajudam no desenvolvimento profissional.

---

## 🎯 Objetivo

Facilitar a conexão entre alunos do 3º CPM e empresas, incentivando a qualificação profissional e auxiliando na inserção dos estudantes no mercado de trabalho.

---

## 👨‍💻 Desenvolvido por

Projeto desenvolvido para o **3º Colégio da Polícia Militar do Paraná (3º CPM)** como iniciativa de apoio aos estudantes na construção de sua carreira profissional.
